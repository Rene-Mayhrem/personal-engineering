# Blog Structure (Optional)

Estructura preparada para publicar contenido técnico en el futuro.

## Carpetas
- `blog/drafts/`: borradores en progreso.
- `blog/published/`: posts listos para publicar.
- `blog/assets/images/`: diagramas e imágenes.

## Convención sugerida
- Archivo: `YYYY-MM-DD-topic-slug.md`
- Frontmatter mínimo:

```yaml
title: "Designing SLOs for Event-Driven Systems"
date: "2026-06-10"
tags: ["sre", "reliability", "architecture"]
summary: "Cómo diseñar SLOs que no generan ruido operativo."
status: "draft"
```

## Recomendación
Publicar solo contenido derivado de artefactos ya sanitizados del repo.
